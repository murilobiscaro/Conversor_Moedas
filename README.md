# 💰 Conversor de Moedas - Rocketseat

Este é um projeto prático desenvolvido como parte dos estudos iniciais de JavaScript na plataforma **Rocketseat**. O objetivo principal foi aplicar conceitos fundamentais da linguagem para criar uma ferramenta funcional de conversão de moedas (Dólar, Euro e Libra) para Real Brasileiro (BRL).

link de acesso: https://murilobiscaro.github.io/Conversor_Moedas/

## 🚀 Tecnologias Utilizadas

- **JavaScript (ES6+):** Lógica de conversão, manipulação de DOM e tratamento de eventos.
- **HTML5:** Estruturação dos elementos do formulário e exibição de resultados.
- **CSS3:** Estilização da interface (baseada no template fornecido).

## ⚙️ Funcionalidades

- **Validação de Input:** O campo de valor aceita apenas números, utilizando expressões regulares (Regex).
- **Conversão em Tempo Real:** Converte valores de USD, EUR e GBP para BRL com base em taxas fixas.
- **Formatação de Moeda:** Utiliza a API `Intl.NumberFormat` para exibir os valores corretamente formatados no padrão brasileiro.
- **Interface Responsiva:** Exibição dinâmica do resultado no rodapé da aplicação.
- **Tratamento de Erros:** Bloco `try...catch` para gerenciar possíveis falhas durante o processo de conversão.

## 🧠 Conceitos Praticados

- Seleção de elementos com `querySelector` e `getElementById`.
- Manipulação de eventos com `addEventListener` e `onsubmit`.
- Uso de `switch` para lógica condicional.
- Template Literals para interpolação de strings.
- Manipulação de classes CSS via JavaScript (`classList`).

## 🛠️ Como Executar

1. Clone este repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Insira o valor, selecione a moeda desejada e clique em converter.
